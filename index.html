如何搭建个人博客（基于GitHub + Hexo)

第一阶段：

1.1 Node.js的安装与配置

点击此处通过官网下载相应版本，无特殊情况默认安装（next就完事）。

win + R 输入 cmd 打开命令提示符， 依次输入 node -v 和 npm -v，若返回版本号，即表示安装成功。

添加国内镜像源：

此时，我们需要配置阿里的镜像进行加速

在命令提示符输入 npm config set registry https://registry.npm.taobao.org 配置淘宝镜像

在命令提示符输入 npm config get registry 检查镜像源是否配置成功

1.2 Git的安装与配置

点击此处通过官网下载相应版本，默认安装。

在命令提示符输入git –version检验安装是否成功

1.3 GitHub账号的注册以及Git仓库的创建

点击此处进入官网 Sign Up 注册账号

点击页面右上角”+“，点击”New repository“进入创建页面

给仓库命名，注意，一定要添加“.github.io”后缀，README的初始化也要勾上，

以及，仓库名必须和账号名保持一致，例如账号名为“Alpha”，则仓库名“Alpha.github.io”

附注，可以选择自己喜欢的主题，在仓库完成创建后，

1.4 Hexo的安装和配置

首先选择一个磁盘新建一个文件夹以存放博客文件，在该文件夹中右键点击 Git Bash Here，

输入 npm install -g hexo-cli 进行安装

输入 hexo -v 检验是否安装成功

Hexo的配置命令，依然是在Git Bash Here中依次输入 hexo init 和 npm install 即可初始化

接下来输入 hexo g 生成静态网页，输入hexo s 运行本地服务器，然后再浏览器打开http://localhost:4000/ 预览博客

显示以下信息说明操作正确：

INFO Hexo is running at http://0.0.0.0:4000/. Press Ctrl+C to stop.

ctrl + c 关闭本地服务器

常用命令：

hexo s (hexo server) 监视文件变动并自动更新，除修改站点配置文件外，无需重启服务器，刷新网页即可生效。

hexo g (hexo generate) 生成静态网页（执行后会在站点根目录下生成public文件夹, hexo会将”/blog/source/“ 下面的.md后缀的文件编译为.html后缀的文件,存放在”/blog/public/ “ 路径下）

hexo d (hexo deploy) 将本地数据部署到远端服务器

hexo cl (hexo clean) 清除缓存，网页正常情况下可以忽略此条命令,执行该指令后,会删掉站点根目录下的public文件夹

附注，Hexo 说明文档

1.5 实现GitHub同本地的连通

生成密钥，仍是右键单击，点击 Git Bash Here 输入以下命名：

git config –global user.name “Name” 你在GitHub的账户名
git config –global user.email “Email” 你在GitHub登记的邮箱

然后生成密钥的命令：

ssh-keygen -t rsa -C “Email” 你在GitHub登记的邮箱

然后出现：

直接回车

然后就是

Your identification has been saved in /c/Users/you/.ssh/id_rsa. Your public key has been saved in /c/Users/you/.ssh/id_rsa.pub. The key fingerprint is:

这里是各种字母数字组成的字符串，结尾是你的邮箱

The key’s randomart image is:

这里也是各种字母数字符号组成的字符串

现在密钥已经生成，输入 clip < ~/.ssh/id_rsa.pub 将密钥复制到剪贴板上

连接GitHub

点击头像，点击 Settings 点击 SSH and GPG keys 新建SSH，建议名字同邮箱

然后将剪贴板上的密钥粘贴进去

本地连接GitHub

右键单击， Git Bash Here 输入 ssh -T git@github.com

使用任意编辑器（可用记事本）打开根目录下的 _config.yml

对最后一段的配置进行修改

通过 Git Bash Here 输入 npm install hexo-deployer-git –save 安装自动部署发布工具

输入 hexo new post “article title” 新建一篇博客
